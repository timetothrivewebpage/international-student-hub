Accessibility

Accessibility goals and current limitations for this student-led research site.

Accessibility approach

This site aims to use clear headings, readable spacing, descriptive links, keyboard focus states, and responsive layouts that work on mobile and desktop screens.

The site is still evolving. Some pages and features may not work as expected yet, and that feedback is useful for improving the experience.

Current limitations

Some pages are still being built, and some features may be incomplete. Please use the feedback form to report anything that is difficult to use, missing, broken, or confusing.
